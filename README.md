# hw-18-budget-tracker

This project we were tasked with creating an application that could create and store various financial records both locally and via a deployed site on heroku via mongoose and atlas db.
# Installation
Mongoose
Atlas DB
Heroku


# Usage
This app could be used to adquately track finances and update your progress as an end user.
